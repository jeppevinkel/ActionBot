# Support

## Bugs
    
You can file a bug report on [GitHub][issues] and tag it as a bug, or on the community [Discord][discord] server, but please make sure to search for existing issues first.
    
## Suggestions

You can post a suggestion on [GitHub][issues] and tag it as an enhancement, or on the community [Discord][discord] server, but please make sure to search for existing suggestions first.

[issues]: https://github.com/BOLL7708/ActionBot/issues
[discord]: https://actionbot.app/discord
